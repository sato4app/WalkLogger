# WalkLogger - 歩行記録アプリ

歩行時の位置情報を記録し、写真を撮影できるPWAアプリケーションです。

## 機能

- **位置情報記録**: Start/Stopボタンで位置情報の記録を制御
- **写真撮影**: カメラで写真を撮影し、サムネイルを表示
- **ログ表示**: 日時とGPS座標（小数点以下5桁）を表示
- **データ保存**: IndexedDBにログと写真を永続化
- **PWA対応**: オフライン動作、インストール可能

## 使い方

### 1. アプリの起動

ローカルサーバーを起動してブラウザで開きます：

```bash
# Pythonを使う場合
python -m http.server 8000

# またはnpx serveを使う場合
npx serve .
```

ブラウザで `http://localhost:8000` を開きます。

### 2. 位置情報の記録

1. **Startボタン**をクリックして記録を開始
2. 位置情報が自動的に記録され、画面左にログが表示されます
3. **Stopボタン**をクリックして記録を停止

### 3. 写真撮影

1. **Photoボタン**をクリック
2. カメラで写真を撮影
3. 撮影した写真のサムネイルが画面右上に表示されます

## アイコン画像について

PWAとして正しく動作させるには、以下のアイコン画像が必要です：

- `icon-192.png` (192x192ピクセル)
- `icon-512.png` (512x512ピクセル)

これらのファイルをプロジェクトのルートディレクトリに配置してください。

### アイコン画像の作成方法

1. お好みの画像編集ツールで192x192と512x512のPNG画像を作成
2. ファイル名を `icon-192.png` と `icon-512.png` にして保存
3. プロジェクトのルートディレクトリに配置

簡易的なアイコンが必要な場合は、オンラインのアイコンジェネレーターを使用することもできます。

## 技術仕様

- **位置情報取得**: Geolocation API（高精度モード）
- **写真撮影**: File API（capture属性使用）
- **データ保存**: IndexedDB
  - `logs` ストア: 位置情報ログ
  - `photos` ストア: 写真データ
- **PWA**: Service Worker + Web App Manifest
- **GPS座標表示**: 小数点以下5桁（約1.1メートルの精度）

## ブラウザ要件

- 位置情報APIに対応したモダンブラウザ
- HTTPS環境（またはlocalhost）での動作
- IndexedDB対応

## データ構造

### logsストア
```javascript
{
  id: (自動生成),
  timestamp: "2024-12-04T12:34:56.789Z",
  latitude: 34.85367,
  longitude: 135.47204
}
```

### photosストア
```javascript
{
  id: (自動生成),
  timestamp: "2024-12-04T12:34:56.789Z",
  imageData: "data:image/jpeg;base64,...", // フル解像度
  thumbnailData: "data:image/jpeg;base64,..." // サムネイル
}
```
